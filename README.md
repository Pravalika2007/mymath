# MathUtilityTool

This Python project is a **Mathematical Utility Tool** using a single reusable class `MyMath`.  
It allows students to perform multiple mathematical operations dynamically based on user input, including:

- Sum of first n natural numbers  
- First n prime numbers  
- Fibonacci series of n numbers  
- Factorial of a number  

---

## 📂 Repository Files

- **`profile.py`** – Main Python program containing the `MyMath` class and all operations.  
- **`output3.txt`** – Sample output generated by the program.  
- **`Output (3).png`** – Screenshot of program output.

---

## ⚙️ Features

- Perform multiple mathematical operations using **one reusable class**  
- Dynamically choose the operation at runtime  
- Works for any value of `n`  
- Easy-to-read console output
